# Kokoro TTS High-speed Pipeline (ONNX Edition)

## 概要
本プロジェクトは、AI音声合成モデル **Kokoro-82M** を **ONNX Runtime (オニキス・ランタイム)** で動作させ、CSVデータから高品質な音声を一括生成するための実行パイプラインです。

各機能を独立させたモジュール設計により、メンテナンス性を高め、個人利用から自動化システムへの組み込みまで柔軟に対応できる構成としています。

## ✨ 主な特徴

* **ONNX Runtimeによる高速動作**:
    * 推論エンジンを最適化し、CPU環境でもリアルタイム以上の生成スピードを実現。
    * マルチスレッド対応により、大量のデータもストレスなく処理可能です。
* **日本語解析の最適化 (misaki / JAG2P)**:
    * `pyopenjtalk` ベースの音素変換により、自然なアクセントと正確な読み上げを実現。
    * アルファベットの読み分け（大文字・小文字の判定）や、記号による「間」の調整に対応しています。
* **スマートリピート機能**:
    * 同一文章を繰り返す際、AI推論を最小限に抑え、波形データの結合で処理を高速化するロジックを搭載しています。
* **整理された内部構造**:
    * UI、データ処理、音声生成の各ロジックを分離。コードの可読性が高く、独自のカスタマイズが容易です。

## 🛠 システム構成

1. **main.py**: 全体のコントロール。
2. **audio_manager_onnx.py**: 音声合成エンジン（ONNX Runtime利用）。
3. **text_processor.py**: テキストの正規化、読み上げルールの適用。
4. **ui_manager.py**: 実行時の対話型インターフェース。
5. **data_manager.py**: CSVデータの抽出と加工。
6. **db_manager.py**: 読み上げ辞書の管理。

## 🚀 セットアップと実行

### 1. 依存ライブラリのインストール
```bash
pip install -r requirements.txt
```
### 2. モデルファイルの配置
以下のファイルをプロジェクト直下に配置してください。

model_quant.onnx (軽量化済みONNXモデル)

voices.bin (ボイスデータ)

### 3. 実行
```bash
python main.py
```
