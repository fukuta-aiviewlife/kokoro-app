# Kokoro TTS Batch Processor

## 概要
本プロジェクトは、AI音声合成モデル **Kokoro-82M** を利用して、構造化されたデータ（CSV）から高品質な音声を一括生成するためのパイプラインです。
テキストの正規化、高度な読み上げルールの適用、およびマルチボイス生成に対応しています。

## 主な機能
- **Text Normalization**: 
  - 記号やアルファベットの読み上げパターンを柔軟に制御（大文字・小文字の判別等）。
  - 文脈に応じたポーズ（タメ）の自動挿入。
- **Data-Driven Workflow**: 
  - CSVの各行から「読み上げ対象」「話速」「ボイス種別」を動的に取得。
  - カラムの欠損に対する自動スキップ処理。
- **WSL-Windows Integration**: 
  - WSL環境で生成した音声を、Windowsホスト側のオーディオデバイスから直接再生・確認するユーティリティを搭載。

## セットアップ
1. **依存ライブラリのインストール**:
   ```bash
   pip install -r requirements.txt
2.設定:
  rules.json を編集することで、固有名詞の読み（辞書）やアルファベットの読み上げマップをカスタマイズできます。   
3.実行方法
  python3 main.py
  実行後、対話型UIを通じて処理対象のデータ範囲や出力モードを選択してください。
4.ファイル構成
  main.py: メインロジック・UI制御
  text_processor.py: テキスト整形・正規化エンジン
  data_manager.py: データ抽出・加工
  audio_manager.py: 音声合成エンジン・再生制御
  rules.json: 読み替えルール設定
